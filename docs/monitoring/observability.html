5️⃣ RECOMMANDATIONS PRIORITAIRES

🔴 [HAUTE] Installer agents LAD/OMS (vm-db, vm-jumpbox)

Pourquoi: Ces VMs ne rapportent aucune métrique de performance. Impossible de monitorer/alerter.

Action:

  • Azure CLI: az vm extension set --vm-name vm-db --name LinuxDiagnostic --resource-group rg-cloudshield-prod
  • Répéter pour vm-jumpbox

Bénéfice: Collecte CPU, mémoire, disque, réseau immédiate après installation (~5-10 min)

🔴 [HAUTE] Vérifier Syslog forwarding

Pourquoi: Syslog table vide. Impossible de monitorer SSH, sudo, erreurs système.

Action:

  • Vérifier rsyslog config: /etc/rsyslog.conf sur chaque VM
  • Ajouter forwarding vers Log Analytics workspace
  • Redémarrer: sudo systemctl restart rsyslog

Bénéfice: Logs de sécurité SSH, sudo, kernel disponibles dans 2-3 min

🔴 [HAUTE] Activer NSG Flow Logs

Pourquoi: NTANetAnalytics table vide. Pas d'analyse du trafic réseau bloqué/autorisé.

Action:

  • Azure Portal → NSG (nsg-prod-web, nsg-prod-app, nsg-prod-db)
  • Flow Logs → Enable → Storage Account + Log Analytics
  • Select workspace: law-cloudshield

Bénéfice: Visibilité complète sur trafic bloqué (DDoS, scan ports, micro-segmentation)

🟡 [MOYENNE] Mettre en place alertes Mémoire > 75%

Pourquoi: vm-web/vm-app stables mais peuvent croître. Alerte proactive nécessaire.

Action:

  • Azure Monitor → Alert Rules → New Alert Rule
  • Condition: Perf table, CounterName = "% Used Memory", > 75%
  • Action Group: Email secops@fintechglobal.local

Bénéfice: Notification immédiate si mémoire dépasse seuil

🟡 [MOYENNE] Monitorer SSH Failed Auth

Pourquoi: Détection brute-force SSH pour War Room (incident response).

Action:

  • Alert Rule: Syslog, "Failed password" or "Invalid user" > 10/hour
  • Scope: Facility = authpriv
  • Action: Email + PagerDuty (incident escalation)

Bénéfice: Réaction rapide aux tentatives d'accès non autorisées

🟡 [MOYENNE] Exporter requêtes automatiquement

Pourquoi: Rétention audit long-terme, rapports récurrents pour conformité.

Action:

  • Créer Logic App ou Function qui exécute requêtes [REQ-01] à [REQ-09] toutes les 6h
  • Exporter résultats vers Blob Storage (format CSV/JSON)
  • Archive → 90 jours retention

Bénéfice: Historique complet pour audits ANSSI/ISO 27001
